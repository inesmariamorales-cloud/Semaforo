Este modo sirve si la pizarra digital no tiene micrófono o si quieres enseñarlo sin permisos.

Cómo subirlo

1. Guarda este archivo con el nombre index.html.

2. Súbelo a GitHub Pages, Netlify o cualquier web con HTTPS.

3. Ábrelo en Safari desde la dirección web.

4. Pulsa Activar micrófono y acepta el permiso.

Importante

Los dB son aproximados. Esta herramienta es visual y educativa.

El micrófono solo funciona bien si la página está abierta desde una web segura, normalmente https://.

Si el navegador o la pizarra no dejan usar el micro, puedes usar el modo Simulación.
